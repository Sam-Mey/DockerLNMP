# DockerLNMP

## 系统要求

### Ubuntu/Debian arm64

## 运行环境

> Nginx 1.25.4
> MariaDB 11.2
> PHP 8.3-fpm  
> Redis 7.2
> phpMyAdmin

## 安装

```bash
curl -O https://raw.githubusercontent.com/Sam-Mey/DockerLNMP/main/install.sh
chmod +x install.sh
./install.sh
```
